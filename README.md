# vve-cubeui

VVE移动端cube-ui脚手架

### 特性
- 基于vue-cli@3工具生成
- 内置cube-ui组件，支持后编译，能够快速使用cube-ui所提供的各样组件进行移动端应用开发
- 内置vue-echarts图形库，能够快速开发echarts各样图形
- 提供Demo.vue样例，通过VVE前端工具生成的代码直接复制黏贴到该文件就可以查看效果

## 开始
```
npm install
```

### 开发
```
npm run serve
```

### 编译
```
npm run build
```

### 代码校验
